# شروعاتي گائيڊ (Getting Started)

Welcome to the **سنڌي (SindhiScript)** programming language! This document covers how to set up your environment to properly write, view, and run Sindhi code.
**سنڌي (SindhiScript)** پروگرامنگ ٻوليءَ ۾ ڀليڪار! هي دستاويز توهان کي اهو سيکاريندو ته پنهنجي ڪمپيوٽر کي سنڌي ڪوڊ لکڻ، ڏسڻ، ۽ هلائڻ لاءِ ڪيئن تيار ڪجي.

## Terminal Setup (ٽرمينل سيٽ اپ)

Because Sindhi is a right-to-left (RTL) script that relies on complex text shaping and specific fonts, it often renders incorrectly (letters disconnected or flipped) in standard terminals like VS Code or Windows CMD out of the box. 
ڇاڪاڻ ته سنڌي ساڄي کان کاٻي لکي ويندي آهي ۽ هن ۾ اکرن جو پاڻ ۾ ڳنڍجڻ ضروري هوندو آهي، عام ٽرمينل جهڙوڪ 'VS Code' يا ونڊوز جي 'CMD' اڪثر سنڌي اکرن کي ٽوڙي يا ابتو ڪري ڏيکاريندا آهن.

To solve this, we have built a **custom Web IDE** specifically designed for SindhiScript.
انهيءَ مسئلي کي حل ڪرڻ لاءِ اسان سنڌي اسڪرپٽ واسطي هڪ **خاص ويب آئي ڊي اي (Web IDE)** تيار ڪيو آهي.

### 1. The Web IDE - Recommended (ويب آئي ڊي اي - تجويز ڪيل)
The Web IDE uses a professional code editor engine with the beautiful Vazirmatn font, a dark theme, and an interactive terminal panel that natively shapes Arabic/Sindhi letters perfectly.
اسان جو 'ويب آئي ڊي اي' هڪ پروفيشنل ڪوڊ ايڊيٽر انجڻ استعمال ڪري ٿو جنهن ۾ خوبصورت Vazirmatn فونٽ ۽ ڊارڪ ٿيم موجود آهن، ۽ هن جي ٽرمينل ۾ عربي/سنڌي اکر مڪمل طور تي سهي ڳنڍجي نظر اچن ٿا.

To start it:
هن کي هلائڻ لاءِ:
1. Open your terminal in the SindhiScript directory. (پنهنجو ٽرمينل سنڌي اسڪرپٽ جي ڊائريڪٽري ۾ کوليو).
2. Run (هلايو): `python web_ide.py`
3. Open your browser to (پنهنجو برائوزر هن لنڪ تي کوليو): `http://localhost:5000`
4. Type your multi-line code and press **Ctrl+Enter** to run it. If your code uses `داخل_ڪرو` (input), you can type directly into the terminal panel at the bottom!
   پنهنجو گهڻين سٽن وارو ڪوڊ لکو ۽ **Ctrl+Enter** دٻائي هلايو. جيڪڏهن توهان جي ڪوڊ ۾ `داخل_ڪرو` (input) جو استعمال ٿيل آهي، ته توهان سڌو هيٺ ڏنل ٽرمينل پينل ۾ جواب لکي سگهو ٿا!

### 2. Terminal Mode - Advanced (ٽرمينل موڊ - ائڊوانسڊ)
If you prefer to use the command line (e.g. `python -m سنڌي`), your terminal **must** support UTF-8 encoding.
جيڪڏهن توهان سڌي طرح ڪمانڊ لائين استعمال ڪرڻ چاهيو ٿا، ته توهان جي ٽرمينل لاءِ UTF-8 سپورٽ ڪرڻ **لازمي** آهي.

- Ensure you install a font that supports the Arabic script block, such as **Vazirmatn** or **Noto Naskh Arabic**, and set it as your terminal font.
  پڪ ڪريو ته توهان ڪو اهڙو فونٽ انسٽال ڪيو آهي جيڪو عربي رسم الخط کي سپورٽ ڪري ٿو، جهڙوڪ **Vazirmatn** يا **Noto Naskh Arabic**، ۽ ان کي ٽرمينل جو فونٽ بڻايو.
- If you are unsure if your terminal is rendering correctly, run the built-in diagnostic tool:
  جيڪڏهن توهان کي پڪ ناهي ته توهان جو ٽرمينل سنڌي سهي ڏيکاري رهيو آهي يا نه، ته اسان جو ڊائگناسٽڪ ٽول هلايو:

```bash
python -m سنڌي --فونٽ_جانچ
```
*(Or type `.فونٽ_جانچ` inside the REPL / يا REPL جي اندر `.فونٽ_جانچ` لکو)*

This will print a Sindhi test string, its raw Unicode codepoints, and a SHA-256 hash. Even if the text on your screen looks disjointed or broken, if the **SHA-256 hash matches**, your environment is correctly processing the bytes — it is purely a visual terminal issue, which is why the Web IDE is recommended.
اهو توهان کي هڪ سنڌي جملو، ان جا يونيڪوڊ ڪوڊ پُوائنٽس ۽ هڪ SHA-256 هيش (hash) ڏيکاريندو. جيڪڏهن توهان جي اسڪرين تي اکر ٽٽل به هجن، پر **هيش (hash)** ملي رهيو آهي، ته ان جو مطلب آهي ته ڪمپيوٽر ٻولي سهي پڙهي رهيو آهي پر ڏسڻ ۾ کيس مسئلو آهي — تنهنڪري اسين ويب آئي ڊي اي استعمال ڪرڻ جي صلاح ڏيون ٿا.

### 3. Roman Transliteration Fallback (`--رومن`)
If you are on a restricted machine where you absolutely cannot install a Sindhi/Arabic font, you can use the Roman fallback mode:
جيڪڏهن توهان ڪنهن اهڙي ڪمپيوٽر تي آهيو جتي توهان ڪو به سنڌي/عربي فونٽ انسٽال نٿا ڪري سگهو، ته توهان رومن موڊ استعمال ڪري سگهو ٿا:

```bash
python -m سنڌي --رومن فائل.سن
```
*(Or type `.رومن` inside the REPL to toggle it / يا REPL ۾ `.رومن` لکي هن کي آن/آف ڪريو)*

This mode transliterates all standard output (`ڇاپ`) and error messages from Sindhi script into Latin (Roman) characters, ensuring you can still read errors and program output. 
هي موڊ توهان جي ڪوڊ جي سموري نتيجي (`ڇاپ`) ۽ غلطين جي نياپن کي رومن سنڌيءَ ۾ ترجمو ڪري ڇڏيندو، ته جيئن توهان پروگرام جون غلطيون ۽ نتيجو سولائي سان پڙهي سگهو.

**Note:** Your source code files (`.سن`) must still be written in pure Sindhi script. This flag only affects the interpreter's output to the terminal.
**نوٽ:** توهان جا ڪوڊ وارا فائل (`.سن`) هر حال ۾ نج سنڌيءَ ۾ لکيل هجڻ گهرجن. هي جھنڊو فقط پروگرام جي نتيجي کي ٽرمينل تي تبديل ڪندو.

For the exact mapping, see [رومن_ترجمو.md](رومن_ترجمو.md).
ان جي مڪمل تفصيل لاءِ [رومن_ترجمو.md](رومن_ترجمو.md) ڏسو.
