# Language Reference (ٻوليءَ جو حوالو)

## 1. Variables and Constants (متغير ۽ ثابت)
- `متغير` (var): Variables can have their values changed later in the program.
  - متغير جي قيمت بعد ۾ پروگرام ۾ تبديل ٿي سگهي ٿي.
- `ثابت` (const): Constants cannot be changed once they are assigned a value.
  - ثابت جي قيمت هڪ ڀيرو طئي ٿيڻ کان پوءِ تبديل نٿي ٿي سگهي.
```sindhi
متغير الف = 10
ثابت ب = 20
```

## 2. Conditionals (شرطيه بيان)
Keywords used: `جيڪڏهن` (if), `ٻي_صورت` (elif), `نه_ته` (else).
استعمال ٿيندڙ لفظ: `جيڪڏهن` (if), `ٻي_صورت` (elif), `نه_ته` (else).
```sindhi
جيڪڏهن الف وڌيڪ 10:
    ڇاپ("وڏو آهي")
ٻي_صورت الف برابر 10:
    ڇاپ("برابر آهي")
نه_ته:
    ڇاپ("ننڍو آهي")
```

## 3. Loops (لوپ)
Keywords used: `جڏهن_تائين` (while), `هر` (for).
استعمال ٿيندڙ لفظ: `جڏهن_تائين` (while), `هر` (for).
```sindhi
جڏهن_تائين ڳڻپ گهٽ 5:
    ڇاپ(ڳڻپ)
    ڳڻپ = ڳڻپ جمع 1

هر ميوو ۾ ميوا:
    ڇاپ(ميوو)
```
- `ٽوڙيو` (break): Used to exit the loop entirely. (لوپ مان مڪمل طور نڪرڻ لاءِ).
- `جاري` (continue): Used to skip to the next iteration of the loop. (لوپ جي ايندڙ ڦيري تي وڃڻ لاءِ).

## 4. Functions (فنڪشن / ڪم)
Keywords used: `ڪم` (function), `موٽايو` (return).
استعمال ٿيندڙ لفظ: `ڪم` (function), `موٽايو` (return).
```sindhi
ڪم ڳڻپ_ڪرو(الف، ب):
    موٽايو الف جمع ب
```

## 5. Modules (ماڊيول امپورٽ ڪرڻ)
Keywords used: `آندو` (import), `مان` (from), `جاريات` (export).
استعمال ٿيندڙ لفظ: `آندو` (import), `مان` (from), `جاريات` (export).
```sindhi
# فائل 1 (رياضي.سن)
ڪم جمع_ڪرو(الف، ب):
    موٽايو الف جمع ب
جاريات جمع_ڪرو

# فائل 2
مان رياضي آندو جمع_ڪرو
ڇاپ(جمع_ڪرو(5، 5))
```

## 6. Object Oriented Programming (او او پي)
Keywords used: `طبقو` (class), `ورثو` (inherits), `پنهنجو` (self), `پيدائش` (constructor).
استعمال ٿيندڙ لفظ: `طبقو` (class), `ورثو` (inherits), `پنهنجو` (self), `پيدائش` (constructor).
```sindhi
طبقو ماڻهو:
    ڪم پيدائش(نالو):
        پنهنجو.نالو = نالو
```

## 7. Operators (آپريٽرز)
- **Math (حسابي)**: `جمع` (+), `گهٽائڻ` (-), `ضرب` (*), `تقسيم` (/), `باقي` (%), `طاقت` (**).
- **Comparison (ڀيٽ)**: `برابر` (==), `نه_برابر` (!=), `وڌيڪ` (>), `گهٽ` (<), `وڌيڪ_برابر` (>=), `گهٽ_برابر` (<=).
- **Logical (منطقي)**: `۽` (and), `يا` (or), `نه` (not).
- **Other (ٻيا)**: `۾` (in).

## 8. Data Types (ڊيٽا جا قسم)
- `عدد` (int), `اعشاري` (float), `متن` (str), `صحيح`/`غلط` (bool), `ڪجهه_نه` (None).
- **List (فهرست)**: `[1, 2, 3]`
- **Dictionary (لغت)**: `{"نالو": "سنسار"}`

## 9. Python Interop - FFI (ٻاهريون ڪم)
The `ٻاهري ڪم` block allows you to execute raw Python code directly inside SindhiScript.
`ٻاهري ڪم` جي ذريعي پائيٿون جو اصلي ڪوڊ سڌو سنڌي اسڪرپٽ جي اندر هلائي سگهجي ٿو.
```sindhi
ٻاهري ڪم وقت_ڏيکاريو():
    import time
    return time.time()
```
