# Roman Transliteration (رومن ترجمو)

The `--رومن` CLI flag and `.رومن` REPL command enable a fallback mode that translates Sindhi script output into Latin (Roman) characters. This is strictly a **one-way translation for terminal output** (e.g. error messages and `ڇاپ` statements) to assist users on systems without Arabic font support. 
`--رومن` ڪمانڊ لائين جھنڊو ۽ REPL جي `.رومن` ڪمانڊ هڪ اهڙو موڊ آن ڪن ٿا، جيڪو سنڌي اسڪرپٽ جي نتيجي کي لاطيني (رومن) اکرن ۾ ترجمو ڪري ڇڏيندو آهي. هي صرف ٽرمينل تي نظر ايندڙ نتيجي (جهڙوڪ `ڇاپ` جا بيان ۽ غلطين جا نياپا) کي رومن ۾ بدليندو آهي ته جيئن جن ڪمپيوٽرن ۾ سنڌي فونٽ ناهي اهي به ڪوڊ جو نتيجو پڙهي سگهن.

Source code must still be written in native Sindhi script.
توهان جو ڪوڊ هر حال ۾ نج سنڌي رسم الخط ۾ لکيل هجڻ گهرجي. هي فقط نتيجي کي رومن ۾ بدليندو.
